# Сервис опроса 

[English version here](README_ENG.md)

## Краткое описание
### Назначение
Cервис позволяет проводить опросы. В сервисе есть три роли: 
- Владелец сервиса
- Создатель опроса
- Респондент

Определения:
- **Владелец сервиса** - физлицо или организация, предоставляющие сервис создания и проведения опросов. 
- **Создатель опроса** - пользователь сервиса, который создаёт опрос(ы), с целью получения ответов и дальнейшего анализа данных.
- **Респондент** - тот, кто отвечает на вопросы опроса. 
- **Сервис** - постоянно работающее приложение, доступное через интернет, с которым можно взаимодействовать через API.

## Техническое описание

Сервис опроса реализован как Spring Boot приложение. 

Приложение: 
- Предоставляет RESTfull API
- Реализует бизнес-логику для управления опросами
- Сохраняет данные в реляционной и документоориентированной базах данных
- Запускается как контейнер в среде исполнения

### Модель развертывания
Сервис контейнеризируется и может быть развернут: 
- В контейнере
- В kubernetes (как объект Deployment/Pod управляющие Pod-ами)
- В облачной среде
